# Deja Tu Huella MVP

MVP de e-commerce para una tienda de indumentaria local en Argentina.

## Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui style components
- lucide-react
- framer-motion
- react-hook-form
- zod

## Cómo correrlo

```bash
npm install
npm run dev
```

Abrí `http://localhost:3000`.

## Scripts

- `npm run dev`
- `npm run lint`
- `npm run build`
- `npm run start`

## Alcance del MVP

- Home
- Catálogo con búsqueda y filtros
- Categorías
- Detalle de producto con variantes
- Carrito persistente en `localStorage`
- Checkout validado
- Contacto
- FAQ
- Política de envíos
- Páginas legales

## Shipping

- Método principal: Correo Argentino
- Mensajería explícita para pueblos y localidades cercanas
- Estimador placeholder preparado para integración futura
